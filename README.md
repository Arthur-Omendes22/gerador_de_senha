# 🔐 Gerador de Senhas Aleatórias

Um script simples e eficiente em **Python** para gerar senhas fortes e seguras de forma automatizada via terminal.

---

## 🚀 Funcionalidades

O programa permite criar senhas personalizadas combinando diferentes tipos de caracteres:

* **Letras Maiúsculas:** (A-Z)
* **Letras Minúsculas:** (a-z)
* **Números:** (0-9)
* **Símbolos Especiais:** (!@#$%^&*_)
* **Personalização de Tamanho:** O usuário define exatamente quantos caracteres a senha deve ter.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [Python 3](https://www.python.org/)
* **Módulo:** `random` (utilizado para garantir a aleatoriedade dos caracteres).

---

## 📋 Como Executar o Projeto

1.  **Pré-requisitos:**
    Você precisa ter o Python instalado em seu computador.

2.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/Arthur-Omendes22/gerador_de_senhas.git](https://github.com/Arthur-Omendes22/gerador_de_senhas.git)
    ```

3.  **Execute o script:**
    ```bash
    python nome_do_seu_arquivo.py
    ```

4.  **Interação:**
    O programa solicitará o tamanho desejado (ex: 12, 16, 20) e exibirá a senha gerada instantaneamente.
